Pepedeulloa's record collection API

Explore My Vinyl Collection with this API

Powering my vinyl record collection website, this API fetches and stores my collection in Discogs data in a database. Seamlessly integrating with the web interface, it offers users an immersive experience exploring my vinyl records.

Technology Stack

Built with Node.js and Express, this API utilizes Turso as the database solution and integrates Swagger for comprehensive documentation. Together, these technologies contribute to a responsive and efficient backend.

API Usage Guidance

Refer to the Swagger documentation for a clear understanding of how to use the API.

Explore the code

For a closer look at the code behind both the API and the web interface, feel free to explore our repositories on GitHub. Dive into the source code, contribute, or simply satisfy your curiosity. Check out the API repository and the web repository.
